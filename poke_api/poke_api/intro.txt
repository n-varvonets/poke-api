CI - цепочка собьтий которая проодит различньіе проверки и подготоваливает приложении к деплою в прод.
Процесс CI:
    1. разраб пушит код на git (pull request)
    2. с помощью action наш git провоидт различньіе проверки (tests)
    3. после проверок данньіе собираются в контенер(creating container)
    4. готовьіе контейнер пушиться в докерхаб(push container)
